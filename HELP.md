# Spring and Vue Monolithic 
 - Develop mode: Spring http://localhost:8080/'den, Vue ise http://localhost:3000/ çalışıyor. 
 - Production : Eğer Maven package edilirse; http://localhost:8080'de Spring ve Vue birlikte çalışacaklardır.
 Burada önemli olan aynı endpointleri iki tarafada vermemek. Spring api altında rest api için end point kullanılmalı. Vue router ile routing yapılmalıdır.
Benze yapı react, angular ya da Flutter ile de kurulabilir.
- Maven:Package ardından " /target" altında oluşturulan jar dosyası herhangi bir ortamda çalıştırılabilir.
